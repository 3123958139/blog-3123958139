### 三、创建工程

    1. CMakeLists.txt
    2. dll_.h
    3. dll_.cpp
        * 以上三个用于创建 c++ dll 封装那些对计算性能要求高的函数，如
        * w_ma_：计算加权平均值

    4. class_.py
        * 封装那些需要重复调用的 py 代码，如
        * get_url_data_：从 tushare 下载数据
        * get_sql_data_：调用 mysql 本地数据
        * run_socket_server_：建立 socket 服务器
        * run_socket_client_：建立 socket 客户端
        * import_dll_：导入动态链接库

    5. test_1.py
        * 用于测试 dll 函数

    6. test_2.py
        * 用于下载 tushare.org 数据
        * 用于调用 mysql 数据
        * 用于运行 socket 服务器

    7. test_3.py
        * 用于运行 socket 客户端
