### **三、创建工程**

    1. CMakeLists.txt
    2. my_cpp_dll_.h
    3. my_cpp_dll_.cpp
        * 以上三个用于创建 c++ dll 封装那些对计算性能要求高的函数，如
        * w_ma_：计算加权平均值

    4. my_py_class_.py
        * 封装那些需要重复调用的 py 代码，如
        * get_url_data_：从 tushare 下载数据
        * get_sql_data_：调用 mysql 本地数据
        * run_socket_server_：建立 socket 服务器
        * run_socket_client_：建立 socket 客户端
        * use_cpp_dll_：调用 c++ 动态链接库

    5. test_py_1.py
    6. test_py_2.py
    7. test_py_3.py
        * 以上三个用于测试代码
